# Generative AI for Beginners (Version 3) - A Course

Master the basics of building Generative AI applications with our comprehensive 21-lesson course by Microsoft Cloud Advocates.

## üå± Getting Started

This course includes 21 lessons. Each lesson focuses on a specific topic, so feel free to start wherever you want!

Lessons are categorized as either "Learn" lessons, which explain a Generative AI concept, or "Build" lessons, which cover a concept along with code examples in both **Python** and **TypeScript** whenever possible.

For .NET developers, check out [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Each lesson also features a "Keep Learning" section with additional resources.
